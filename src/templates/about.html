About the Deep Learning Project

This project leverages Deep Learning techniques using PyTorch to classify images with high accuracy. The system is powered by the ResNet-50 architecture.

Supports both image uploads and URL-based predictions.

Project Highlights:

  • Model: ResNet-50 (pretrained on ImageNet)
  • Framework: PyTorch
  • Deployment: Flask-based Web Interface
  • Checkpoint Path: models/resnet50_checkpoint.pth
  • Outputs: Predicted class, confidence, and analysis
