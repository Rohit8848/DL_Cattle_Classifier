This project is on Cattle Breed Recognition System using Deep Learning Resnet50 architecture. It is trained on 26 indian cows breed and 6 buffaloes breed.
This project has a User Interface that takes the image or link of breed of cattle and make a prediction on the breed of the cattle. The UI is built using Flask framework.

Citation- Ajit Kumar Singh, Data Scientist, Indian cattle breed images 2020
Dataset link-https://drive.google.com/u/0/uc?id=1nSN9SpDJEAHe4pCmKWy0X7oCPyx4BrjT&export=download


Use Case :
-reduces the time for the prediction of breed and don't need an expert everytime you buy and sell the cattle and provides a simple interface
